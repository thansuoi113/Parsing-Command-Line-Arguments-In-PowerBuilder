# Parsing-Command-Line-Arguments-In-PowerBuilder
Parsing Command Line Arguments In PowerBuilder

Arguments Parsing In PowerBuilder.
Ex: parsecmd.exe -A Avalue -B Bvalue -C Cvalue
	parsecmd.exe --A Avalue --B Bvalue -C Cvalue
	parsecmd.exe /A Avalue /B Bvalue /C Cvalue
	parsecmd.exe -A "Avalue" --B Bvalue /C "Cvalue"
